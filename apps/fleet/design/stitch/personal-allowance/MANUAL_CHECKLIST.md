# Personal Allowance — Manual test checklist

1. **Policy off** — Tier Config → Personal Allowance disabled → recon company/driver shares match pre-feature behavior for the same week.
2. **100% quota** — Enable policy; weekly quota met; measured 100 km Personal → $0 overage; company absorbs full Personal fuel $.
3. **50% quota** — Under 60% band → 0 earned → all Personal $ on driver (overage).
4. **Finalize once** — Ledger deductions match new `driverShare`; re-finalize warning still works; optional next-week bonus only when top band + bonus km > 0.
5. **Mobile settings** — Tier Config Personal Allowance tab usable on phone (stacked bands, one Save).
